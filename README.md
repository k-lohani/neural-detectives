# Noir Deductions

A voice-powered murder mystery game where you interrogate suspects, deduce clues, and solve crimes alongside Detective Louis — a grizzled AI detective voiced in real-time by Google Gemini.

Built with React, FastAPI, MongoDB, and Google's Gemini + Imagen AI models.

## How It Works

1. **Pick a difficulty** — Easy (3 suspects), Medium, or Hard (5 suspects)
2. **Read the case briefing** — a noir-styled comic strip sets the scene
3. **Talk to Detective Louis** — speak or type to discuss suspects, weapons, and locations
4. **Deduce the truth** — the logic grid updates automatically as you reason through clues
5. **Make your accusation** — name the WHO, WHAT, and WHERE before time runs out

## Features

- **AI Case Generation** — every mystery is procedurally generated with Gemini 2.5 Pro, ensuring unique puzzles each time
- **Voice Interaction** — speak to Detective Louis using your microphone; he responds with a noir-grizzled voice via Gemini TTS
- **Noir Icon Art** — suspect portraits, weapon icons, and location illustrations generated by Imagen 4 in a consistent black-and-white ink style
- **Comic Strip Intros** — each case opens with a 4-panel noir comic strip generated in the background
- **Constraint-Propagation Logic Grid** — a Murdle-style deduction grid that auto-updates based on your conversation
- **Focus Agent** — mention a suspect or entity and the UI automatically navigates to their card
- **Hint System** — ask Louis for hints (costs you score points)
- **Scoring** — rated out of 100 based on time, hints used, and wrong accusations
- **Session Persistence** — transcripts, grid state, and scores saved to MongoDB
- **Auto-Accusation** — when the grid resolves to a unique solution, the accusation fields auto-populate

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React 18, Vite 5, React Router v6 |
| Backend | Python, FastAPI, Uvicorn |
| Database | MongoDB (Motor async driver) |
| AI Models | Gemini 2.5 Pro, Gemini 2.5 Flash, Imagen 4 Fast, Gemini TTS |
| Voice | Web Speech API (recognition), Gemini TTS (synthesis) |
| Transport | REST (case creation) + WebSocket (real-time gameplay) |

## Getting Started

### Prerequisites

- Python 3.9+
- Node.js 18+
- MongoDB (local or Atlas)
- Google Cloud project with Vertex AI enabled (or a Gemini API key)

### Backend

```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Environment Variables

Create a `.env` file in the project root:

```
GOOGLE_GENAI_USE_VERTEXAI=true
GOOGLE_CLOUD_PROJECT=your-project-id
GOOGLE_CLOUD_LOCATION=us-central1
MONGODB_URI=mongodb+srv://...
```

Then open `http://localhost:5173` to play.
